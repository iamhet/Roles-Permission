# Step for Installation

## Step 1 :Key Generate
```php artisan key:generate  ```

## Step 2 : Copy .env file
```cp .env.example .env  ```

## Step 3 : Run Migration Command
```php artisan migrate ```

## Step 4 : Run Seed
```php artisan db:seed  ```
```php artisan db:seed --class=PermissionTableSeeder  ```

## Step 5 :
```php artisan serve ```
